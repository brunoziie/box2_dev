Box2 v0.2b
============

Box2 is a Javascript MVC Framework for PhoneGap Android Applications based
in Spaghetti* PHP framework.

This framework already includes:
- Cordova plugin version 1.5.0
- Handlebars 1.0.rc.1
- iScroll 
- jQuery 1.8.2 (default desabled)
